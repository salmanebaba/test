AutoShifter: Automated Gear Shifting System

A smart IoT solution that automates gear shifting based on real-time pedaling cadence. Using an ESP32 microcontroller and servo motor, the system dynamically adjusts bicycle gears to maintain optimal cycling efficiency. Features cadence monitoring, and a lightweight design, reducing physical strain while enhancing riding performance. Developed with a focus on embedded systems and energy-efficient IoT integration.
